Coins

Platform: Android Studio​

Coins is an android app, which is designed to help college students to manage their money conveniently. As we known, college students are often the people without much money to control. But they also want to buy some things expensive, like a SLR camera or a high performance computer. So this app can let user to set a goal, which is the amount of money you want to get and the time you like to spend. It help user to form a habit of reasonable consumption and accumulate the money they save everyday to achieve their goals.

One belt one road

Platform: Visual Studio2015 ArcGIS​

"One belt one road" is a WebGIS developed based on ArcGIS API for Microsoft Silverlight. The purpose of this application is to let people learn about the plan of one belt one road from the perspective of map. Actually, the route of this plan and related information can be clearly seen on this platform. The main functions of this application include road map, display of the features of different countries along the road, and some basic map functions, like zoom in, zoom out, navigation and so on.
